# 🤖 Toolyfi Automation Bot

**Free, Open-source GitHub Actions + Telegram Bot for Website Automation**

Automatically deploy, validate SEO, check articles, detect errors, and get instant notifications!

---

## ✨ Features

- ✅ **GitHub Actions** — Auto-validate on code push
- ✅ **Telegram Bot** — Real-time notifications & commands
- ✅ **SEO Checker** — Validate meta tags, titles, descriptions
- ✅ **Article Validator** — Check article quality & word count
- ✅ **Sitemap Generator** — Auto-generate robots.txt & sitemap
- ✅ **Error Detection** — Catch bugs before deployment
- ✅ **Build Logs** — Monitor validation status
- ✅ **100% Free** — No hidden costs
